# Proyecto MongoDB - Estudiantes

Autor: Kevin Luengas

Este proyecto es un programa en Python que permite manejar estudiantes en una base de datos usando MongoDB. Se puede agregar, ver, actualizar y eliminar estudiantes desde un menú en consola.

Para la base de datos usé MongoDB Atlas. Ahí creé un cluster, configuré un usuario y saqué la cadena de conexión para poder conectar el programa desde Python. También activé el acceso para poder conectarme sin problemas.

La base de datos que utilicé se llama BD_CursoMongo y la colección Estudiantes. Desde el programa se van guardando los datos que uno ingresa.

También usé MongoDB Compass para ver los datos de forma más visual. Me conecté con la misma cadena de conexión y ahí pude revisar que los estudiantes sí se estaban guardando, además de verlos, borrarlos o modificarlos directamente.

El programa funciona con un menú donde uno elige qué hacer. Probé agregando datos de compañeros, listándolos, cambiando edades y eliminando algunos para verificar que todo funcionara bien.

En general, el ejercicio me ayudó a entender cómo conectar Python con MongoDB y cómo manejar datos en una base de datos NoSQL de forma básica.
